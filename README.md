# Dundunpildun ⚽

**Dundunpildun** is a web app for creating randomized World Cup 2026 team draws among friends. Add participants, select teams, and run an animated draw with confetti, music, and shareable results — perfect for friendly World Cup watch parties.

Built with [Next.js](https://nextjs.org).

## Features

- **Setup** — Add friends and pick which World Cup 2026 teams to include
- **Animated Draw** — Watch teams get randomly assigned with a spinning animation, progress bar, and confetti
- **Results** — View per-friend team assignments, copy a shareable link, and generate Instagram-style story images
- **Language** — Supports Bahasa Indonesia and English

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
